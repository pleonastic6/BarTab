# BarTab

Lokale Android-Tablet-App für einfachen Getränkeverkauf.

## Ziel

BarTab ist kein offizielles Kassensystem, sondern eine lokale Verkaufs-App für einen Abendbetrieb mit:
- großen Touch-Flächen
- Getränkekategorien
- bearbeitbarem Warenkorb
- lokaler Historie
- Storno / Wiederöffnen
- Produktpflege direkt in der App

## Geplanter Stack

- Kotlin
- Jetpack Compose
- Navigation Compose
- Room
- Material 3

## Kernscreens

- Verkauf
- Historie
- Produkte
- Verkaufsdetail / Bearbeiten

## Status

Aktuell: lokale Projektplanung und technisches Grundgerüst.
