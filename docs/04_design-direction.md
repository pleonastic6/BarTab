# Design Direction

## Grundidee
- dunkles UI
- große, klare Touch-Zonen
- wenig visuelles Rauschen
- Fokus auf Geschwindigkeit und Lesbarkeit

## Farbrollen
- Anthrazit als Basis
- Grün für aktive / erfolgreiche Zustände
- Orange für Warnung
- Rot für Fehler / Storno
- helle Neutrals für Text und Karten

## Typografie
- funktionale Sans Serif für UI
- optional Character-Font nur für Branding oder große Überschriften
