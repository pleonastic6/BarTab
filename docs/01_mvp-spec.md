# MVP Spec

## Kernfunktionen
- Getränke über große Buttons auswählen
- Warenkorb live bearbeiten
- Verkauf abschließen und lokal speichern
- Verkäufe in Historie ansehen
- Verkäufe wieder öffnen oder stornieren
- Produkte und Kategorien lokal pflegen

## Nicht Teil von V1
- Online-Sync
- Benutzerkonten
- Bondruck
- Fiskal-/TSE-Integration
- Multi-Device-Sync

## Plattform
- Android Tablet
- offline-first
